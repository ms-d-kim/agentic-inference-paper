→ 1-page PM exec summary

The non-obvious point: the literal user of an inference service is increasingly the agent itself — a machine client emitting a long, correlated, stateful request stream — not the humans/orgs "using agents," who sit behind it and never touch the serving contract. NVIDIA inference sells directly to ≤3 of these segments; the rest are demand-shapers, SLO-setters, or standards levers. Quantitative magnitudes are directional unless cited; Dynamo's 85–97% is realized single-session under its own routing (motivation only).
